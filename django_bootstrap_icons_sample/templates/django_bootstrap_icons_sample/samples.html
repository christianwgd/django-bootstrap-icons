Material Design Icons will be resized to same base size

Left is a Bootstrap Icon (base size 16) Right is a Material Design icon (base size 20, original size is 24)

{% bs_icon 'alarm' extra_classes="bi-valign-default" %}{% md_icon 'alarm' extra_classes="bi-valign-default" %} default


If you want to have the same size as Bootstrap Icons set them to a size factor of 1.25
If Bootstrap Icon is of size 2em, set Material Icon to 2.5em.
Material Design Icons have some internal "padding" in svg so you need to include the django-bootstrap-icons css file to make some adjustments to the alignment
